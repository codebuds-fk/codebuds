Streamlining Handoff with Figma Dev Mode

Shamrooz Gul

The handoff from design to development is a pivotal moment that can make or break product build cycles. Getting it right accelerates iteration and alignment. Getting it wrong leads to confusion and disjointed experiences. In this talk, I will demonstrate how Figma's Dev Mode streamlines handoff by allowing designers to create development specifications, resources and guidance directly within the Figma file.

WASM in FK Ads

Abinash Mohapatra

Video ads are made of a video element and an image creative as shown above. The video files could go up to a few 100MBs in size. Nonetheless, the file has to be uploaded but we’d still need to run a few video health checks before uploading them to our servers, just like we did for image files. These checks include aspect ratio, bitrate, accepted audio decibel levels etc. (and more to be added with time). JavaScript does not provide too much tooling to run these checks out of the box. So, ideally these checks would need to run on the server, which uses ffmpeg. But going over the entire process of file upload (as explained above) just to find out that a certain check failed seemed a clear wastage of time and user’s (and our) resources. So we wanted to see if we could run these checks “before” and only upload the file if we are sure the file is safe.
