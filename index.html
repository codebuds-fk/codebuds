An Approach for Monorepo in React Native Apps

Mathan Appadurai

Monorepo for web projects have been there for quite a while and it is quite popular among many of the big projects like React, Babel etc., So why not extend the same for react-native projects also as react native projects are also primarily the same (except the native dependencies), and get the same advantage as single linting, build setup and better dev experience and collaboration. In my talk I will talk about an approach on doing the same.
